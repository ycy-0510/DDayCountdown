# DDayCountdown

A simple and elegant macOS menu bar application that helps you count down to important exam dates.

## Features

- **Menu Bar Countdown**: Always visible D-Day countdown in your menu bar.
- **Exam Selection**: accurate countdowns for multiple exam schedules.
- **Auto-Update**: Fetches the latest exam dates from a remote source.
- **Launch at Login**: Option to automatically start the app when you log in.
- **Dark Mode Support**: Seamlessly integrates with macOS system appearance.

## Requirements

- macOS 15.0 or later
- Xcode 16.0+ (for building)

## Installation

1. Clone the repository.
2. Open `DDayCountdown.xcodeproj` in Xcode.
3. Build and run.

## Dependencies

- [LaunchAtLogin-Modern](https://github.com/sindresorhus/LaunchAtLogin-Modern): For adding "Launch at Login" functionality.
